unsigned long WINAPI GetMySpecificBytes(unsigned int hSocket, unsigned char* pBuffer, unsigned long ulLen);

 

Parameters

 

hSocket

            [in]A peer socket handle

 

pBuffer

            [in]A pointer to a pre-allocated buffer

 

ulLen

            [in]Size of the buffer (pBuffer)

 

Remarks

 

            A client may send a server an array of specific bytes of data by calling the method IUSocket::SendMySpecificBytes. Once you get such a base request on server side, you could use this method to retrieve an array of specific bytes.

 

            The function returns the number of actual bytes retrieved. You may need to call the function GetCountOfMySpecificBytes to query the number of specific bytes.
