typedef void (CALLBACK *POnSSLEvent) (unsigned int hSocket, int nWhere, int nRtn);

 

Parameters

 

hSocket

            [in]A peer socket handle

 

nWhere

            [in]A data (tagSSLEvent) indicating SSL/TLS handle-shake being in process

 

nRtn

            [in]Some other data

 

Remarks

           

The callback notifies your code that SSL/TLS handshake is in process now.

 

This event always happens within main thread.
