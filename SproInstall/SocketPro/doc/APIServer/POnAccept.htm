typedef void (CALLBACK *POnAccept) (unsigned int hSocket, int nError);

 

Parameters

 

hSocket

            [in]A peer socket handle

 

nError

            [in]A network communication error code

 

Remarks

           

The callback notifies your code that a client initializes a socket connection.

 

Specifically if nError is 10061, the peer socket is rejected because one client may build too many socket connections. Refer to the property MaxConnectionsPerClient.

 

This event always happens within main thread.
