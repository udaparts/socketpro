bool WINAPI InitSocketProServer(int nParam);

 

Parameter

 

nParam

            [in]A reserved parameter

 

Remarks

           

            Calling this function starts an instance of SocketPro server at the very beginning before calling any other functions. At the very end, the function UninitSocketProServer is expected to be called for cleaning all of previously allocated resources

 

            The class CSocketProServer constructor calls this function internally so that you should not call this function again.
