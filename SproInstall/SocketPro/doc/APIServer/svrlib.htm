Chat/notification service – SocketPro provides a built-in service notification (chat) service. Once a client establishes a socket connection, the service is ready to use after switching for any service other than start up service. With this built-in service service, you can send a message from either client or server side to one or more groups of other clients. Alternatively, you can also send a private message to one specific client. Notification service is very useful under many cases.

Listening socket and others – A SocketPro server supports one listening socket only. However, a server can host any number of services and reusable libraries. SocketPro supports SSL and TLS to secure socket communication. Before calling any functions inside this library, the function InitSocketProServer must be called. If you use SocketProAdapter, you shouldn’t call this function because this function is already wrapped inside the constructor CSocketProServer. A number of functions are provided to tune up SocketPro server for properties, but all of them have default values. Therefore, usually there is no need to touch them.
