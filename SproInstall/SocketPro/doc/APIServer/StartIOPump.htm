bool WINAPI StartIOPump();

 

Remarks

 

            Calling this function after the function StartSocketProServer starts IO completion port pump to drive non-blocking socket events.

 

            If you have specified using window message pump instead of IO completion port pump to drive non-blocking socket events, the function will return false. Otherwise, if the function does not return true, you can use window system function GetLastError to query error code.
