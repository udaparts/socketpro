bool WINAPI SendUserMessageEx(unsigned int hSocket, const wchar_t *strUserID, const unsigned char *pMessage, unsigned long ulLen);

 

Parameters

 

hSocket

            [in]A peer socket handle or client going to send a private message

 

strUserID

            [in]A user id string

 

pMessage

            [in]A pointer to an array of bytes to be sent

 

ulLen

            [in]The size of pMessage in byte

 

Remarks

           

You can use the function to send a private message from a client (hSocket) to a client (strUserID) on server side.

 

Note that similarly you can also call the method IUChat::SendUserMessageEx on client side to send a private message to a user.

 

If SocketPro is not able to find the peer socket hSocket or strUserID is a null pointer or an empty string, the function will return false.

 

Refer to the methods SendUserMessage, SpeakTo and SpeakToEx.
