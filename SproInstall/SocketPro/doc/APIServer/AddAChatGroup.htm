bool WINAPI AddAChatGroup(unsigned long ulGroupID, const wchar_t *strDescription);

 

Parameters

 

ulGroupID

            [in]An identification number to represent a chat group

 

strDescription

            [in]A brief description to a chat group (ulGroupID)

 

Remarks

           

            Calling this function adds a chat group id (usGroupID) into SocketProServer with its brief description.

           

SocketPro always supports chat service for all of peer sockets as long as a peer socket is not switched to the startup service sidStartup. A chat group id must be 1, 2, 4, 8, 16, 32, and so on. Because an unsigned long has 32 bits, SocketPro chat service only supports 32 groups at most.

 

The function will return false if a chat group id is already added.
