unsigned long WINAPI GetAChatGroup(unsigned long ulGroupID, wchar_t *strDescription, unsigned long ulChars);

 

Parameters

 

ulGroupID

            [in]An identification number to represent a chat group

 

strDescription

            [out]A buffer for receiving a brief description for the chat group (ulGroupID)

 

ulChars

            [in]Size of the buffer (strDescription) in char

 

Remarks

           

            Calling this function retrieves a brief chat group description with a given chat group id. The function returns the number of actually returned chars.

           

If a group is not found, the function returns 0.
