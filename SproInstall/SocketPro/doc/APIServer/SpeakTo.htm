bool WINAPI SpeakTo(unsigned int hSocket, const VARIANT *pvtMsg, unsigned int hSocketTarget);

 

Parameters

 

hSocket

            [in]A peer socket handle or client going to send a private message

 

pvtMsg

            [in]A pointer to a message

 

hSocketTarget

            [in]A peer socket handle or client receiving the private message (pvtMsg)

 

Remarks

           

You can use the function to send a private message from a client (hSocket) to a client (hSocketTarget) on server side.

 

Note that you can also call the method IUChat::SpeakTo on client side to send a private message.

 

If SocketPro is not able to find both peer sockets hSocket and hSocketTarget or pvtMsg is a null pointer, the function will return false.
