bool WINAPI SendUserMessage(unsigned int hSocket, const wchar_t *strUserID, const VARIANT *pvtMsg);

 

Parameters

 

hSocket

            [in]A peer socket handle or client going to send a private message

 

strUserID

            [in]A user id string

 

pvtMsg

            [in]A pointer to a VARIANT that contains a message to be sent

 

Remarks

           

You can use the function to send a private message from a client (hSocket) to a client (strUserID) on server side.

 

Note that you can also call the method IUChat::SendUserMessage on client side to send a private message to a user.

 

If SocketPro is not able to find the peer socket hSocket or either strUserID or pvtMsg is a null pointer, the function will return false.

 

Refer to the methods SendUserMessageEx, SpeakTo and SpeakToEx.
