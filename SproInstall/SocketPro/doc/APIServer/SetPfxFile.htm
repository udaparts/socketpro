void WINAPI SetPfxFile(const wchar_t *strPfxFile, const wchar_t *strPassword);

 

Parameters

 

strPfxFile

            [in]A full file name protected by a password (strPassword) in the pfx format for SSL/TLS certificate and key.

 

strPassword

            [in]A password

 

Remarks

 

            Calling this function sets a password and a full file name in the pfx format for SSL/TLS certificate and key. The file is protected by a password (strPassword). Later, a SocketPro server will use them for securing socket communication. Once they are not needed any more, the previously set password will be cleaned inside the SocketPro server as early as possible.
