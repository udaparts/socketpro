typedef HRESULT (CALLBACK *PSLOW_PROCESS) (unsigned short usRequestID, unsigned long ulLen, unsigned int hSocket);

 

Parameters

 

usRequestID

            [in]A fast request ID

 

ulLen

            [in]Size of request parameters in byte

 

hSocket

            [in]A peer socket handle

 

Remarks

           

The callback notifies your code that a slow request has arrived and been dispatched to a worker thread.

 

This event always happens within a worker thread.
