bool WINAPI Enter(unsigned int hSocket, unsigned long ulGroups);

 

Parameters

 

hSocket

            [in]A peer socket handle

 

ulGroups

            [in]One or more chat groups

 

Remarks

           

Calling this method joins one or more chat groups (ulGroups) on server side.

 

You can also call the method IUChat::Enter on client side to join chat groups.

 

If SocketPro is not able to find the peer socket hSocket or none of expected chat groups exits, the function will return false.
