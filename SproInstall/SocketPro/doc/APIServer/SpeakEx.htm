bool WINAPI SpeakEx(unsigned int hSocket, const unsigned char *pMessage, unsigned long ulLen, unsigned long ulGroups = odAllGroups);

 

Parameters

 

hSocket

            [in]A peer socket handle or client going to send a private message

 

pMessage

            [in]A pointer to a message

 

ulLen

            [in]Size of message buffer (pMessage) in byte

 

ulGroups

            [in]One or more group ids that are bit-wise or-ed

 

Remarks

           

You can use the function to send a public message from a client (hSocket) to one or more groups of clients on server side.

 

You can also call the method IUFast::SpeakEx on client side to send out a public message.

 

If SocketPro is not able to find both peer sockets hSocket, the function will return false.
