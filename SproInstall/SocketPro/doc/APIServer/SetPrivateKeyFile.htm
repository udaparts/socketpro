void WINAPI SetPrivateKeyFile(const wchar_t *strPrivateKeyFile);

 

Parameter

 

strPrivateKeytFile

            [in]A string pointing to a private key file in the openSSL format pem

 

Remarks

           

            In addition to MS SSPI, SocketPro supports openSSL to secure data communication. To use openSSL with encryption method set to either SSL23 or TLSv1, you have to call this method to set this file before calling the function StartSocketProServer.
