bool WINAPI StartBatching(unsigned int hSocket);

 

Parameter

 

hSocket

            [in]A peer socket handle

 

Remarks

           

            Calling the method starts batching return results into a memory buffer so that we could send all of requests onto a client using the least number of packets later and make sure the network bandwidth is best utilized.

 

            SocketPro provides this special feature to batch a number of small return results, but don’t use the feature to batch a large size of return result, which may cost too much for coping result from an input buffer into internal memory buffer.
