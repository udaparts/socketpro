bool WINAPI SetPassword(unsigned int hSocket, const wchar_t *strPassword);

 

Parameters

 

hSocket

            [in]A peer socket handle

 

strPassword

            [in]A password

 

Remarks

 

            Calling this function sets a password within the callback function OnIsPermitted, only if a SocketPro server uses UDAParts security protocol for securing data communication between client and server.
