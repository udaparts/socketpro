unsigned long WINAPI SendReturnData(unsigned int hSocket, unsigned short usRequestId, unsigned long ulLen, const unsigned char* pBuffer);

 

Parameters

 

hSocket

            [in]A peer socket handle

 

usRequestId

            [in]A request identification number

 

ulLen

            [in]Size of buffer (pBuffer)

 

pBuffer

            [in]A buffer containing return result

 

Remarks

           

Calling this function sends a return result onto a client. The function returns the actual number of bytes of the result.

 

In addition, the function may return special data like SOCKET_NOT_FOUND, REQUEST_CANCELED and RETURN_DATA_INTERCEPTED. If the return data is SOCKET_NOT_FOUND, it indicates that the peer socket (hSocket) is already closed or going to be closed. If the return data is REQUEST_CANCELED, the peer socket gets a special request IUSocket::Cancel to stop processing the current request. If the return data is RETURN_DATA_INTERCEPTED, the return result is intercepted.

 

Note that the function SendReturnData is able to take care of slow client connections automatically, if the function is called from a worker thread (slow request).
