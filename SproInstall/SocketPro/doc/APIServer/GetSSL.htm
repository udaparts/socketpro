void* WINAPI GetSSL(unsigned int hSocket);

 

Parameter

 

hSocket

            [in]A peer socket handle

 

Remarks

           

            The method returns a pointer to either an openSSL structure SSL or a MS SSPI structure CtxtHandle.

           

            SocketPro exposes internal data structures for your special needs. However, usually you shouldn’t use them directly. In case you have special requirements, you can use these internal structures cautiously.

 

            The function will return false if hSocket is not found on server side.
