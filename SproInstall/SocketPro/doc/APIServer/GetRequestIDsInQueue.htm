unsigned long WINAPI GetRequestIDsInQueue(unsigned int hSocket, unsigned short *pusRequestID, unsigned long ulSize);

 

Parameters

 

hSocket          

[in]A peer socket handle

 

pusRequestID

            [in]A pre-allocated buffer for receiving an array of request IDs to be processed in queue.

 

ulSize

            [in]Size of the buffer (pusRquestID)

 

Remarks

 

            Calling this function queries how many requests to be processed in queue. The function returns the number of actually obtained requests in queue.
