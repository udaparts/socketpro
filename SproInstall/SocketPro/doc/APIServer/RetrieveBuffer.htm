unsigned long WINAPI RetrieveBuffer(unsigned int hSocket, unsigned long ulLen, unsigned char* pBuffer, bool bPeek = false);

 

Parameters

 

hSocket

            [in]A peer socket handle

 

ulLen

            [in]Size of buffer (pBuffer) in byte

 

pBuffer

            [out]A pre-allocated buffer for receiving request data

 

bPeek

            [in]A boolean data indicating if data is copied into the buffer (pBuffer) but is not removed from internal memory queue

 

Remarks

           

Calling this function copies the input data of a request into a pre-allocated memory buffer (pBuffer).

 

If the input parameter bPeek is false, the input data of the request will be removed from internal memory queue.

 

However, if bPeek is true, the request data is copied into the memory buffer but is not removed from internal memory queue.

 

Note that you should set bPeek to false usually. You can only set the parameter to true if you have a particular requirement. Without removing the current request data from internal memory queue, SocketPro will not dispatch the next request.
