void WINAPI SetPingInterval(unsigned long ulPingInterval);

unsigned long WINAPI GetPingInterval();

 

Remarks

           

            Calling the two methods sets and queries the property PingInterval. By default, it is 60,000 ms.

 

            SocketPro will ping a remote client for abnormal termination if there is no data movement over a socket connection for a time period longer than this property.
