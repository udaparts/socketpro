[helpstring("method SendUserMessageEx")] HRESULT SendUserMessageEx([in]BSTR bstrUserID, [in]unsigned long ulLen, [in]BYTE *pMessage);

 

Parameters

 

bstrUserID

            [in]A string for user id

 

ulLen

            [in]The size of message buffer in byte

 

pMessage

            [in]A buffer of message to be sent

 

Remarks

 

            Similar to the method SpeakToEx, calling this method sends a user (bstrUserID) an array of bytes.

 

            Refer to the methods SpeakTo and SpeakToEx.
