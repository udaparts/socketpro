[id(6), helpstring("method Send")] HRESULT Send([in]VARIANT vtData, [out, retval]long *plSent);

 

Parameters

 

vtData

            [in]A VARIANT data contains an array of bytes data

 

plSent

            [out]A pointer to an integer for receiving the actual number of bytes sent

 

Remarks

           

Calling the function sends an array of bytes data onto a remote non-SocketPro server.

 

Note that you must set the property UseBaseSocketOnly to true before calling the function. Otherwise, the function will return the error code uecUnexpected.

 

The function is written for communicating to a non-SocketPro server only.
