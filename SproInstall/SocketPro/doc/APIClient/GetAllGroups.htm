[id(95), helpstring("method GetAllGroups")] HRESULT GetAllGroups();

 

Remarks

 

            Calling the function sends a base request to a remote SocketPro server for querying all of groups supported within the server.

 

            Once the request is processed and returned, you can use the property Groups to get the total number of opened chat groups and the method GetGroupInfo to retrieve each of groups information.
