[id(3), helpstring("method LockASocket")] HRESULT LockASocket([in]long lTimeout, [in]IUSocket *pIUSocketSameThreadWith, [out, retval]IUSocket **ppIUSocket);

 

Parameters

 

lTimeout

            [in]A timeout in ms

 

pIUSocketSameThreadWith

            [in]An interface pointer to an USocket object

 

ppIUSocket

            [out]An pointer to an interface of IUSocket for receiving an interface to a locked USocket object

 

Remarks

 

Calling this function locks a USocket object within a given timeout in ms. If the locking is successful, *ppIUSocket will be an interface to a USocket object and the USocket object reference is increased by 1. If failed, *ppIUSocket will be NULL.

 

If pIUSocketSameThreadWith is not NULL, a locked USocket object shares the same thread with the USocket object pointed by pIUSocketSameThreadWith.

 

Note that you should unlock a locked USocket object as soon as possible so that the locked USocket object can be returned into the socket pool and reused by others.
