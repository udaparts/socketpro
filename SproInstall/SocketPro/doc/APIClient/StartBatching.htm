[id(32), helpstring("method StartBatching")] HRESULT StartBatching();

 

Remarks

 

            Calling this function starts batching a set of requests.

 

            SocketPro provides the important feature for improving network communication efficiency by batching all of requests into as a small number of packets as possible.

 

            SocketPro doesn’t support recursive batching. Therefore, you can only call this method again after calling the method CommitBatching or AbortBatching.
