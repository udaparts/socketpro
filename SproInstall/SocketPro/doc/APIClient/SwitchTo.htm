[id(31), helpstring("method SwitchTo")] HRESULT SwitchTo([in]long lSvsID);

 

Parameter

 

lSvsID

            [in]A service identification number

 

Remarks

 

            Calling this function sends a request to ask for a service identified by its id (lSvsID).

 

            This call leads transferring client credentials (user id and password) and a structure of CSwitchInfo about client information onto a remote SocketPro server.

 

            You must call this function for all of services first before calling other methods.
