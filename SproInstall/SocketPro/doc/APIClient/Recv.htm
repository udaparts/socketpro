[id(7), helpstring("method Recv")] HRESULT Recv([out, retval]VARIANT *pvtData);

 

Parameter

 

pvtData

            [out]A pointer to a VARIANT data structure for receiving an array of bytes data

 

Remarks

           

Calling the function receives an array of bytes data from a remote non-SocketPro server. Afterwards, you need to make sure the received data is released by calling window system function VariantClear.

 

Note that you must set the property UseBaseSocketOnly to true before calling the function. Otherwise, the function will return the error code uecUnexpected.

 

The function is written for communicating to a non-SocketPro server only.
