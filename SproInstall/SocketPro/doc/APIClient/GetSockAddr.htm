[id(15), helpstring("method GetSockAddr")] HRESULT GetSockAddr([out]long *plPort, [out, retval]BSTR *pbstrSockAddr);

 

Parameters

 

plPort

            [out]A pointer to an integer for receiving point number.

 

pbstrSockAddr

            [out]A pointer to BSTR for receiving IP address like 111.222.121.212

 

Remarks

 

            Calling the function retrieves the local ip address like 111.222.101.211 for a socket.

 

Refer to socket function getsockname.
