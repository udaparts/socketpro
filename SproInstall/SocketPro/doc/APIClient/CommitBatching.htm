[id(33), helpstring("method CommitBatching")] HRESULT CommitBatching([in, defaultvalue(0)]VARIANT_BOOL bSvrBatching);

 

Parameter

 

bSvrBatching

            [in]A boolean data sent to a remote SocketPro server to indicate if batching return results is required

 

Remarks

 

            Calling this function commits previously batched requests and sends them onto a remote SocketPro server with a least number of network packets. If bSvrBatching is true, the SocketPro server will batch returned results first and then send them to a client. If bSvrBatching is false, return results are not batched.
