[id(10), helpstring("method OnRequestProcessed")] HRESULT OnRequestProcessed([in]long hSocket, [in]short nRequestID, [in]long lLen,  [in] long lLenInBuffer, [in]short sFlag);

 

Parameters

 

hSocket

            [in]A client socket handle

 

nRequestID

            [in]A request id

 

lLen

            [in]Size of a request result in byte

 

lLenInBuffer

            [in]Actual number of bytes of a request result available in buffer

 

sFlag

            [in]A flag of the enum tagReturnFlag

 

Remarks

 

The event arrives once SocketPro detects that a return result of a request (nRequest) is coming from a remote SocketPro server.

 

            When the flag sFlag is set to rfCompleted, your must call the method either GetRtnBufferEx or GetRtnBuffer to retrieve the returned result for processing unless both lLenInBuffer and iLen are zero.
