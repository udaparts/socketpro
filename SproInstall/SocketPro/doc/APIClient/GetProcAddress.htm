[id(14), helpstring("method GetProcAddress")] HRESULT GetProcAddress([in]BSTR bstrOpenSSLFuncName, [out, retval]long *plProcAddr);

 

Parameters

 

bstrOpenSSLFuncName

            [in]A function name in openSSL libraries.

 

pProcAddr

            [out]A pointer to an integer for receiving openSSL function pointer address

 

Remarks

 

            This function returns a function address of openSSL libraries from a given function name (bstrOpenSSLFuncName).

 

            SocketPro wraps all of openSSL functions for simplifying your development. However, you may want to use particular features from openSSL libraries directly, you can call this function to get and cautiously use them.
