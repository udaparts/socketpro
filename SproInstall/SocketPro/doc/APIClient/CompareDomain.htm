[id(201), helpstring("method CompareDomain")] HRESULT CompareDomain([out]BSTR *pbstrDomain, [out, defaultvalue(0)]BSTR *pbstrHost, [out, retval]VARIANT_BOOL *pbMatchable);

 

Parameters

 

pbstrDomain

            [in]A pointer to a BSTR for receiving a domain string

 

pbstrHost

            [in]A pointer to a BSTR for receiving a host string

 

pbMatchable

            [in]A pointer to a boolean for receiving if the host (pbstrHost) is matchable to the subject string of a certificate

 

Remarks

 

            Compare a remote server domain name with a certificate subject string. If the domain name does not match against the certificate subject string, you must be very cautious for the sake of security.

 

            After calling the method, you will also get a domain (pbstrDomain) and host name (pbstrHost).

 

            Note that the method may be very expensive at very first call for a host because it internally calls socket function gethostbyaddr if a client host IP address is in the format like “111.222.212.121”.

 

            Note that *pbMatchable will be always true if both client and server run on the same machine.
