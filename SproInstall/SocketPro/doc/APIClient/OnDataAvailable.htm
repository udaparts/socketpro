[id(1), helpstring("method OnDataAvailable")] HRESULT OnDataAvailable([in]long hSocket, [in]long lBytes, [in] long lError);

 

Parameters

 

hSocket

            [in]A client socket handle

 

lBytes

            [in]The number of bytes available in socket receiving buffer

 

lError

            [in]An error code

 

Remarks

 

The event arrives when client socket detects data coming from a remote server.

 

            Note that the event may happen if the property UseBaseSocketOnly is set to true or the property ReturnEvents is set with the flag rfDataComing if the property UseBaseSocketOnly is set to false.
