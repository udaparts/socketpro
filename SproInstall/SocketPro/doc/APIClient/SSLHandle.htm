[propget, id(21), helpstring("property SSLHandle")] HRESULT SSLHandle([out, retval] long *pVal);

 

Remarks

 

This property indicates a SSL handle.

 

If the property EncryptionMethod is either SSL23 or TLSv1, the handle is a pointer to openSSL SSL structure.

 

If the property EncryptionMethod is either MSSSL or MSTLSv1, the handle is a pointer to the structure CtxtHandle within MS SSPI.

 

SocketPro wraps all of SSL and TLS features for simplifying your development. SocketPro exposes the two structures because you may need special features from either openSSL or MS SSPI. However, you must be highly cautious when directly accessing these structures.
