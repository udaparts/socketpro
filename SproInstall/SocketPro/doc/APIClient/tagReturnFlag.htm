enum tagReturnFlag

{

            rfComing = 1, //A return result is coming

            rfReceiving = 2, //Receiving return result

            rfDecrypted = 4, //Encrypted result is decrypted by Blowfish

            rfUnzipped = 16, //Zipped result is unzipped

            rfCompleted = 32, //return result has arrived completed

            rfDataComing = 64, //raw socket data is coming

};

 

Remarks

 

            The enum contains a set of states of receiving data from a SocketPro server.

 

            Refer to the property ReturnEvents.
