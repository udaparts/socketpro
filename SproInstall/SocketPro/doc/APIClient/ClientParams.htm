[propget, id(63), helpstring("property ClientParams")] HRESULT ClientParams([out, retval] VARIANT *pVal);

[propput, id(63), helpstring("property ClientParams")] HRESULT ClientParams([in] VARIANT newVal);

 

Remarks

 

            Calling the two methods queries and sets an array of client parameters onto the internal structure CSwitchInfo, which will be transferred onto a remote SocketPro server when calling the method SwitchTo.

 

            You will always get five parameters, but you can only set the last three parameters of the structure CSwitchInfo because the first two parameters are reserved.

 

            Refer to the structure CSwitchInfo.
