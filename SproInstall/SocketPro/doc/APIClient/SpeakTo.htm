[id(94), helpstring("method SpeakTo")] HRESULT SpeakTo([in]BSTR bstrIPAddr, [in]long lPort, [in]VARIANT vtMsg);

 

Parameters

 

bstrIPAddr

            [in]IP address in the format like “111.222.212.121”

 

lPort

            [in]A port number that a client socket is running on

 

vtMsg

            [in]A message

 

Remarks

 

            Calling the function sends a private message onto a client identified by its IP address (bstrIPAddr) and a port number (lPort).

 

            To send a private message to a client, you may need to get a set of clients from a remote SocketProServer by sending the request either GetAllListeners or GetAllClients.

 

            The method supports ip addresses "255.255.255.255" (ip address none) and "0.0.0.0" (ip address any). When a client sends a message to one of the two IP addresses, SocketPro server will immediately reflect the message back to the client without regularly seeking a socket session.

 

            Note that the destination client doesn't have to join a chat group.

 

            Refer to the method SpeakToEx.
