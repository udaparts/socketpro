[id(98), helpstring("method GetGroupInfo")] HRESULT GetGroupInfo([in]long lIndex, [out]long *plGroup, [out, retval]BSTR *pbstrDescription);

 

Parameters

 

lIndex

            [in]Zero-based index

 

plGroup

            [in]A pointer to an integer for receiving a chat group id

 

pbstrDescription

            [in]A pointer to a BSTR for receiving a brief description for the chat group

 

Remarks

 

            Calling the function queries a chat group information after sending the request GetAllGroups.
