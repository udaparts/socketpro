[propget, id(69), helpstring("property Frozen")] HRESULT Frozen([out, retval] VARIANT_BOOL *pVal);

[propput, id(69), helpstring("property Frozen")] HRESULT Frozen([in] VARIANT_BOOL newVal);

 

Remarks

 

            This property Frozen determines if window graphic user interfaces are frozen. If the property is true, window user interfaces are disabled when the property Syn is set to true or method WaitAll or Wait is called. Otherwise, window user interfaces are not disabled if a client socket is established within GUI thread.

 

            If a client socket is not established from window user interface thread, window user interfaces will be always frozen if the property Syn is set to true or either WaitAll or Wait is called

 

            By default, this property is false.
