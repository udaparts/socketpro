[helpstring("method SendRequestEx")] HRESULT SendRequestEx([in]unsigned short usRequestID, [in]unsigned long ulLen, [in, size_is(ulLen)]BYTE pBuffer[]);

 

Parameters

 

usRequestID

            [in]A request id

 

ulLen

            [in]Size of the buffer pBuffer in byte

 

pBuffer

            [out]A buffer containing a request input data

 

Remarks

 

            Calling the function sends a request to a remote SocketPro server.

 

            SocketProAdapter and all of client COM objects use this function to send a request to a remote SocketPro server for better performance.

 

            Refer to the function SendRequest.
