[id(16), helpstring("Method GetBytesReceived")] HRESULT GetBytesReceived([out, defaultvalue(0)]long *plHigh, [out, retval] long *plLow);

 

Parameters

 

plHigh

            [out]A pointer to an integer for receiving high 32-bit of data.

 

plLow

            [out]A pointer to an integer for receiving low 32-bit of data

 

Remarks

 

Calling the function retrieves the number of bytes of data having been received since a socket connection is established.

 

USocket library
