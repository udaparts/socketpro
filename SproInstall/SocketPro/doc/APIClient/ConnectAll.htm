[id(5), helpstring("method ConnectAll")] HRESULT ConnectAll([in]BSTR bstrHost, [in]long lPort, [in]long lInitialSvsID, [in]BSTR bstrUID, [in]BSTR bstrPWD, [in]short EncryptionMethod, [in, defaultvalue(-1)]VARIANT_BOOL bEnableZip);

 

Parameters

 

bstrHost

            [in]An IP address to a remote SocketPro server

 

lPort

            [in]The port number of the listening socket of a remote SocketPro server

 

lInitialSvsID

            [in]An initial service id

 

bstrUID

            [in]A user id that will be transferred onto a remote SocketPro server for authentication

 

bstrPWD

            [in]A password that will be transferred onto a remote SocketPro server for authentication

 

EncryptionMethod

            [in]A data indicating encryption method

 

bEnableZip

            [in]A boolean value indicating if online zipping is set to true on both client and server sides

 

Remarks

 

Calling this method connects all of sockets in a socket pool to a remote SocketPro server and immediately asks for the service lInitialSvsID after connection.

 

Note that you can call the method Connect to establish socket connection one by one instead.
