[id(10), helpstring("method GetPeerName")] HRESULT GetPeerName([out]long *plPeerPort, [out, retval]BSTR *pbstrPeerName);

 

Parameters

 

plPeerPort

            [out]A pointer to an interger for receiving peer port number

 

pbstrPeerName

            [out]A pointer to a BSTR for receiving the address of peer like “111.222.212.121”

 

Remarks

 

            Calling this function gets the peer socket address and port. Refer to socket function getpeername.
