[id(9), helpstring("method GetInterfaceAttributes")] HRESULT GetInterfaceAttributes([out]long *plMTU, [out]long *plMaxSpeed, [out]long *plIType, [out]BSTR *pbstrMask, [out, retval]BSTR *pbstrDesc);

 

Parameters

 

plMTU

            [out]A pointer to an integer for receiving MTU (Maximum Transfer Unit)

 

plMaxSpeed

            [out]A pointer to an integer for receiving the maximum speed in bits per second like 10,000,000, 100, 000,000, or 1000,000,000.

 

plType

            [out]A pointer to an integer for type of network device. It can be one of values in the enum tagInterfaceType.

 

pbstrMask

            [out]A pointer to a BSTR for receiving network set mask like “255.255.255.0”.

 

pbstrDesc

            [out]A pointer to a BSTR for receiving network card description

 

Remarks

 

            Calling the function gets network card or modem interface attributes.

 

For example, you can use these attributes to estimate the maximum speed and turn on zip on server side on the fly.
