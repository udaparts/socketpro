[id(80), helpstring("method Wait")] HRESULT Wait([in]short sRequestID, [in]long lTimeout, [in, defaultvalue(0)]long lSvsID, [out, retval]VARIANT_BOOL *pbTimeout);

 

Parameters

 

sRequestID

            [in]A request id

 

lTimeout

            [in]A timeout in ms

 

lSvsID

            [in]A service id

 

pbTimeout

            [out]A pointer to a Boolean value for receiving if the request is timeout

 

Remarks

 

                Similar to the method WaitAll, calling this function wait until the request (sRequestID) of a given service (lSvsID) is returned and its return result is processed.

 

            If lTimeout is –1, the function will be never timeout until the request is processed on server side and its result is returned.

 

            Note that the boolean value pbTimeout will be set to false if a socket connection is broken.
