[id(3), helpstring("method Shutdown")] HRESULT Shutdown([in, defaultvalue(hsSend)]long lHow);

 

Parameter

 

lHow

            [in] A flag (hsSend, hsReceive or hsBoth ) that describes what types of operation will no longer be allowed.

 

Remarks

 

            Calling the function disables sending or receiving data on a socket. Also, its SocketPro server will be notified that a client is about to close a connection. SocketPro server will close the socket connection. At the end, the client gets the event of OnSocketClosed because its server closes the socket connection.

 

The function gracefully closes a socket connection.

 

            Refer to the function Disconnect.
