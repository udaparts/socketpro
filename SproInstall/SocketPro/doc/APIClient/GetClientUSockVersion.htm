[id(50), helpstring("method GetClientUSockVersion")] HRESULT GetClientUSockVersion([out]short *pnMinor, [out,retval]short *pnMajor);

 

Parameters

 

pnMinor

            [out]A pointer to an integer for receiving minor version number

 

pnMajor

            [out] A pointer to an integer for receiving major version number

 

Remarks

 

            Calling this function queries the SocketPro protocol version information on client side.

 

            At this time, both minor and major versions are set to 0.
