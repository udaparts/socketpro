[helpstring("method GetRtnBufferEx")] HRESULT GetRtnBufferEx([in]unsigned long ulLen, [in, size_is(ulLen)]BYTE pBuffer[], [out, retval]unsigned long *plRtnSize);

 

Parameters

 

ulLen

            [in]Size of the buffer pBuffer in byte

 

pBuffer

            [out]A pointer to a previously allocated buffer

 

plRtnSize

            [out]A pointer to an integer for receiving the number of bytes received actually

 

Remarks

 

            Calling the function retrieves an array of bytes of the return result of the current request. The function returns the number of bytes received actually.

 

            SocketProAdapter and all of client COM objects use this function to retrieve returned results from a remote SocketPro server for better performance.

 

            Refer to the function GetRtnBuffer.
