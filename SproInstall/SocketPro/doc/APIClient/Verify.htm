[id(202), helpstring("method Verify")] HRESULT Verify([out]long *plErrorCode, [out, retval]BSTR *pbstrErrorMsg);

 

Parameters

 

plErrorCode

            [in]A pointer to an integer for receiving an error code

 

pbstrErrorMsg

            [in]A pointer to a BSTR for receiving an error message

 

Remarks

 

            Calling this method verifies a peer certificate against a previously stored certificate file that may contain multiple certificates for openSSL or a certificate stored in window certificate store for MS SSPI.

           

Refer to the property VerifyLocation.
