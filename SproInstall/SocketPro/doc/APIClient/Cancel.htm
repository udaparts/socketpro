[id(40), helpstring("method Cancel")] HRESULT Cancel([in, defaultvalue(odCancelAll)]long lRequests);

 

Parameter

 

lRequests

            [in]The number of requests to be canceled

 

Remarks

 

            Calling the method tries to cancel a given number (lRequests) of queued requests to be processed. If lRequests is –1, SocketPro will try the best to cancel all of queued requests.

           

            The actual number of requests canceled is indicated by the property RequestsCanceled.

 

            SocketPro provides this special feature to cancel a long lasting request under special cases without closing and reconnecting a socket connection to a remote SocketPro server so that all of state data are kept unchanged in the SocketPro server.
