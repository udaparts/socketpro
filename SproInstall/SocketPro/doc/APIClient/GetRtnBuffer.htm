[id(39), helpstring("method GetReturnedBuffer")] HRESULT GetRtnBuffer([in]long nLen, [out, retval]VARIANT *pvtData);

 

Parameters

 

nLen

            [in]The number of bytes to be retrieved

 

pvtData

            [out]A pointer to a VARIANT structure for receiving an array of bytes

 

Remarks

 

            Calling the method retrieves an array of bytes for the current request. The actual number of bytes will never be larger than nLen.

 

            Because the method requires you to create a VARIANT data at run time, it is slower than the method GetRtnBufferEx. Therefore, it is not recommended that you use the function unless your development environment supports automation only.
