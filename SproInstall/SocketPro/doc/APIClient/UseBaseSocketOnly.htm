[propget, id(23), helpstring("property UseBaseSocketOnly")] HRESULT UseBaseSocketOnly([out, retval] VARIANT_BOOL *pVal);

[propput, id(23), helpstring("property UseBaseSocketOnly")] HRESULT UseBaseSocketOnly([in] VARIANT_BOOL newVal);

 

Remarks

 

This property determines if you use the methods Send and Recv to exchange data between a client and a non-SocketPro server.

 

      If setting the property to false, you establish a socket connection to a remote SocketPro server. Therefore, you can not use the above two methods to exchange data with a SocketPro server.

 

            Refer to methods Send and Recv.
