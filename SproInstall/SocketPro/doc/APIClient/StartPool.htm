[id(1), helpstring("method StartPool")] HRESULT StartPool([in]BYTE bThreadCount, [in]BYTE bSocketsPerThread);

 

Parameters

 

bThreadCount

            [in]The number of threads with a socket pool

 

bSocketsPerThread

            [in]The number of sockets hosted within a thread

 

Remarks

 

Calling this function starts a socket pool that has a given number (bThreadCount) of threads. Each of the pool threads supports a given number (bSocketsPerThread) of sockets.

 

If bThreadCount is zero, internally bThreadCount will be automatically set to the number of processors.

 

Note that bThreadCount * bSocketsPerThread must not be over 64.
