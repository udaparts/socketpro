[id(52), helpstring("method DetectNetPerformance")] HRESULT DetectNetPerformance([out, retval]short *psNetPerformance);

 

Parameters

 

pnMajor

            [out]A pointer to an integer for receiving network performance

 

Remarks

 

            The function detects a real network bandwidth approximately. The returned result is one of the enum valuse of tagNetPerformance.

 

            Note that it is cheap to call the function if the network is phone-line modem. Otherwise, it is very expensive to call this function. Therefore, the function should not be called whenever possible.
