Client side socket library -- USocket.dll

 

USocket.dll is a COM library for managing socket communication on client side.

 

It contains two ATL/C++ COM objects. The first COM object USocket exposes a set of interfaces, IUSocket, ISocketBase, IUChat and IUFast as well as event interface _IUSocketEvent. In addition, you can get an interface IUCert through the property PeerCertificate if you use SSL/TLS to secure socket communication.

 

All of other client code or software components must be written from the COM object USocket.

            The second COM object is USocketPool exposing one interface IUSocketPool. The COM object is created for easily managing a pool of sockets that are hosted within a few threads. The object is created mainly for web and middle tier (server) applications development.
