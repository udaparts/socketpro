[id(2), helpstring("method OnOtherMessage")] HRESULT OnOtherMessage([in]long hSocket, [in]long nMsg, [in]long wParam, [in]long lParam);

 

Parameters

 

hSocket

            [in]A client socket handle

 

nMsg

            [in]A message

 

wParam

            [in]The first parameter

 

lParam

            [in]The second data

 

Remarks

 

The event arrives after a client starts client SSL/TLS handshake.

 

            You can use the event to track client SSL/TLS handshake progress. If the parameter nMsg is equal to msgSSLEvent, the parameter wParam will be one of members within the enum tagSSLEvent.

 

            The third parameter can be ignored.
