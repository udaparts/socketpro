[id(35), helpstring("method WaitAll")] HRESULT WaitAll([in]long lTimeout, [out, retval]VARIANT_BOOL *pbTimeout);

 

Parameters

 

lTimeout

            [in]A timeout in ms

 

pbTimeout

            [out]A pointer to a boolean value for receiving if the call is timeout

 

Remarks

 

            Calling this function wait until all of queued requests are processed and results are returned.

 

            If lTimeout is 0, the function actually checks if there is a request to be processed. If lTimeout is –1, the function will be never timeout until all of requests are processed and their results are returned.

 

            Note that the boolean value pbTimeout will be set to false if a socket connection is broken.
