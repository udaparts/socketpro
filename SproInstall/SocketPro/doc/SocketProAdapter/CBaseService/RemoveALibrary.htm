public static bool RemoveALibrary(string strLibFile)

public static bool RemoveALibrary(System.IntPtr hInstance)

 

Parameters

 

strLibFile

            [in]A string pointing to a previously added library in a SocketPro server

 

hInstance

            [in]A handle to a previously added library in a SocketPro server

 

Remarks

 

Calling this method posts a message into a SocketPro server for removing a library from a SocketPro server and its associated services and peer sockets. If required, you should call the two methods within main thread.

 

Refer to the native methods RemoveADll and RemoveADllByHandle.
