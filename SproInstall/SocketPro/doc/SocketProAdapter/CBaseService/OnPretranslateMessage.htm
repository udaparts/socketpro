protected virtual bool OnPretranslateMessage(int hWnd, int nMessage, int wParam, int lParam, int uTime, int nPointX, int nPointY)

 

Parameters

 

hWnd

            [in]A window handle

 

nMessage

            [in]A window message

 

wParam

            [in]The first parameter

 

lParam

            [in]The second parameter

 

uTime

            [in]Time tick count at this time

 

nPointX

            [in]The x-value of a point

 

nPointY

            [in]The y-value of a point

 

Remarks

 

This virtual function is called within a worker thread when the worker thread receives a message from main thread or other threads. If you returns true, the worker thread will not continue to process this message. Otherwise, the worker thread will continue to process this message.

 

This virtual function is not often used. If you need the function, use it with caution.

 

For details, refer to the native callback PPRETRANSLATEMESSAGE and window message structure MSG.
