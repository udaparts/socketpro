public abstract class CBaseService

 

            This class mainly wraps the structure CSvsContext and manages a set of its associated CClientPeer-derived objects. Also, this class contains a set of functions for managing reusable libraries.

 

The class exposes a set of protected virtual (callback) functions, but all of them are not often used.

 

An instance of CBaseService-derived can be thought as a manager of its associated CClientPeer-derived objects. You must override the protected abstract function GetPeerSocket to associate a CClientPeer-derived with an instance of CBaseService-derived.
