public USOCKETLib.tagZipLevel ZipLevel { set; get; }

 

Remarks

 

This read/write property indicates one of three compression levels (zlDefault, zlBestSpeed and zlBestCompression) on server side for a socket connection. 

 

Refer to the native property ZipLevel as well as client request SetZipLevelAtSvr.

 

Member of SocketProAdapter.ServerSide.CClientPeer
