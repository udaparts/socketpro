protected virtual void OnReleaseResource(bool bClosing, int nInfo)

 

Parameters

           

bClosing

            [in]A Boolean indicating if a peer socket is going to be closed

 

nInfo

            [in]An error code or new service id

 

Remarks

 

This virtual function is called within main thread when a peer socket is going to be closed or a client asks for a new service by calling the method IUSocket::SwitchTo. You may need to release allocated resources by overriding this virtual function.

 

Refer to the native callback POnClose and structure CSvsContext.
