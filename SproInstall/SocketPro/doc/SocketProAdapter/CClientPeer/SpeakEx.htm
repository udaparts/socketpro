public bool SpeakEx(byte[] pMessage, int nGroups)

public bool SpeakEx(System.IntPtr pMessage, int nLen)

public bool SpeakEx(System.IntPtr pMessage, int nLen, int nGroups)

 

Parameters

           

pMessage

            [in]A buffer containing message to be sent

 

nGroups

            [in]One or more chat groups that are bit-wise or-ed. It defaults to all of chat groups (-1 or 0xFFFFFFFF)

 

nLen

            [in]Size of the buffer pMessage in byte

 

Remarks

 

Calling the function sends a message to one or more chat groups from server side.

 

Refer to the native methods SpeakEx and IUFast::SpeakEx.
