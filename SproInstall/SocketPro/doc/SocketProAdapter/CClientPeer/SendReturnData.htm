Parameters

            [in]A buffer for returning data

            [in]A request id

 

nLen

            [in]An integer indicating the size of the buffer pBuffer in byte

            [in]A memory buffer containing returning result

 

hSocket

            [in]A peer socket handle

 

Remarks

 

Calling the function sends returning result to a client after processing a request. The function returns the number of actually sent data in byte. In addition, the function may return the special data like SOCKET_NOT_FOUND, REQUEST_CANCELED and RETURN_DATA_INTERCEPTED.

 

For details, refer to the native method SendReturnData.
