public bool GetInterfaceAttributes(ref int nMTU, ref int nMaxSpeed, ref tagInterfaceType Type, ref int nMask)

 

Parameters

 

nMTU

            [out]An integer for receiving MTU (Maximum Transfer Unit)

 

nMaxSpeed

            [out]An integer for receiving maximum speed in bits per second

 

Type

            [out]A data for receiving interface type

 

nMask

            [out]An integer for receiving network subnet mask like “255.255.255.0”

 

Remarks

 

Calling the function returns network card interface attributes.

 

Refer to the native method GetInterfaceAttributes.
