protected virtual bool OnSendReturnData(short sRequestID, int nLen, System.IntPtr pBuffer)

 

Parameters

           

sRequestID

            [in]A request id

 

nLen

            [in]Size of returning data buffer pBuffer in byte

 

pBuffer

            [in]A buffer containing returning result

 

Remarks

 

This virtual function is called when sending a returning result to a remote client by calling the method SendReturnData. This virtual function is rarely used.

 

If the function returns true, the returning result is intercepted and method SendReturnData will return RETURN_DATA_INTERCEPTED.

 

Refer to the native callback POnSendReturnData and structure CSvsContext.
