SocketProAdapter

 

            SocketProAdapter is a namespace containing a set of classes and child namespaces that mainly abstract all of complexities of direct use of raw SocketPro native APIs.

           

  1. CClientSocketException – A simple class that wraps client socket communication error
  2. CCriticalSection – A simple class that encapsulates window object CRITICAL_SECTION
  3. CSocketProServerException – A simple class that represents an exception transferred from a SocketPro server
  4. CUPerformanceQuery – A simple class that wraps window high-resolution performance counter
  5. CUQueue – A memory queue that dramatically simplifies packing and unpacking various types of data
  6. IUSerializer – An interface for implementing customer serialization and de-serialization

 

ClientSide – A sub namespace containing a few classes for easily managing client socket, socket pool, and client request and response handler.

 

  1. CClientSocket – A class for wrapping the client COM object USocket
  2. CAsyncServiceHandler – A class for sending requests and handling results
  3. CSocketPool<THandler> -- A generic class for quickly and easily taking advantage of COM object USocketPool

 

ServerSide – A sub namespace containing a few classes for simply managing listening socket, chat and customer services, and client requests.

 

  1. CBaseService – An abstract class for managing a set of CClientPeer-derived objects
  2. CClientPeer – An abstract class for easily processing fast and slow requests from a client
  3. CSocketProServer – A class for easily creating a listening socket for accepting all of incoming socket connections

 

There are two versions of SocketProAdapter source codes available in the directories ..\UDAParts\SocketPro\SproAdapter\dotNet2 and ..\UDAParts\SocketPro\SproAdapter\dotNet11 for .NET 2 and .NET 1.1, respectively. Both versions are compactable to each other. However, SocketProAdapter for .NET 2 is faster. There is a version of SocketProAdapter for pocket pc inside the directory ..\UDAParts\SocketPro\SproAdapter\SproAdapterCe. All of versions of SocketProAdapter are provided with source code. You are encouraged to compile them by yourself for your debugging.

 

            Source files of SocketProAdapter for C++ are available in the directory ..\UDAParts\SocketPro\include. They are sprowrap.cpp and sprowrap.h. Within C++ development environment, you can simply include the two files into your project.

 

           This documentation is created on SocketProAdapter for .NET 2. Others are similar but may have a minor difference.
