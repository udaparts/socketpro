public void SetHeadPosition()

 

Remarks

 

            Calling this function sets head position to 0, and moves packed data in buffer if needed.

 

Without calling this function, the underlying memory may keep growing and the property HeadPosition keeps larger and larger if you call the methods Push, Pop, PushXXX and PopXXX repeatedly but you have never unpacked all of packed data. Normally, this case will not happen. If this particular case does happen, you can call this method to prevent the underlying memory from increasing.
