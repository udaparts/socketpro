public int PopVariantBool(ref bool b)

 

Parameter

           

b

            [out]A boolean value for receiving automation boolean data

 

Remarks

 

            Calling this function unpacks a VARIANT Boolean (COM automation Boolean) value. Afterwards, the VARAIN Boolean is converted into a .NET Boolean. The function returns the actually obtained data size in byte.
