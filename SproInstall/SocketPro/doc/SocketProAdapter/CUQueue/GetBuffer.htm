public System.IntPtr GetBuffer()

public System.IntPtr GetBuffer(int nPos)

 

Parameter

           

nPos

            [in]Offset position of packed buffer. By default, it is 0

 

Remarks

 

            Calling this function returns a pointer to the underlying native buffer offset by a position nPos. If the property Size is 0, the function returns a pointer to the head position of underlying buffer. If the nPos is larger than the property Size, the function returns a pointer to the end of underlying buffer.
