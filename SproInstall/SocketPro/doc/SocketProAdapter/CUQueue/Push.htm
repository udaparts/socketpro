obData

            [out]An object. It can be null.

 

bToNative

            [in]A Boolean. If the input is true, the boxed datetime and bool are converted into COM VARIANT date and bool, respectively before packing. By default, it is true.

 

BDecToCY

            [in]A Boolean. If the input is true, the boxed decimal is converted into COM currency before packing. By default, it is true.

 

USerialize

            [in]An object exposing the interface IUSerialize. USerialize can NOT be null.

 

ptr

            [in]A pointer to a buffer

 

Remarks

 

            Calling one of these functions packs a data into the underlying buffer of an instance of CUQueue.

 

            Refer to the method Save for packing a string.
