public void SetUID(string strUID)

 

Parameter

 

strUID

            [in]A user identification

 

Remarks

 

Calling this method sets a user id into a COM USocket object so that the data can be transferred to a remote SocketPro server for authentication when calling the method SwitchTo.

 

Refer to the native property UserID and method SwitchTo of the interface IUSocket.

 

Member of SocketProAdapter.ServerSide.CClientSocket
