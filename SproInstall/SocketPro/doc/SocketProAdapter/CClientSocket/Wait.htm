public bool Wait(short sRequestID)

public bool Wait(short sRequestID, int nTimeout)

public bool Wait(short sRequestID, int nTimeout, int nSvsID)

 

Parameters

 

sRequestID

            [in]A request identification number

 

nTimeout

            [in]A timeout value in ms. It defaults to infinite

 

nSvsID

            [in]A service identification number. It defaults to the current service id

 

 

Remarks

 

Calling this method waits for a specified request (sRequestID) to be processed and returned with a specified timeout (nTimeout) of one service (nSvsID).

 

The function will return true if the specified is indeed processed and returned before the specified timeout. Otherwise, it returns false. If a socket is closed, the function also returns false.

 

Refer to the method Wait of the interface IUSocket.
