public void DisableUI()

public void DisableUI(bool bDisable)

 

Parameter

 

bDisable

            [in]A Boolean value determining if window graphical user interface is disabled. It defaults to true.

 

Remarks

 

Calling this method disables client graphic user interfaces. If graphic user interfaces are disabled, graphic user interfaces will be frozen when calling the method Wait or WaitAll, or setting the property Syn to true.

 

Refer to the methods Wait and WaitAll as well as the property Syn of the interface IUSocket.
