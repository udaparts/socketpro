            [in]A service identification number

            [in]A Boolean value indicating if exception on server side is automatically transferred onto a client side

            [in]A reference to an asynchronous handler

 

Remarks

 

Calling this method asks for a specified service. If the bAutoTransferServerException is set to true, client may receive a server side exception. See the property SocketProServerException.

 

For details, refer to the method SwitchTo of the interface IUSocket.
