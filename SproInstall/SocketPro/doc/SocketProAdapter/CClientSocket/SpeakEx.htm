public bool SpeakEx(int nGroups, SocketProAdapter.CUQueue UQueue)

public bool SpeakEx(byte[] MessageBuffer, int nGroups)

public bool SpeakEx(int nGroups, System.IntPtr pBuffer, int nLen)

 

Parameter

 

nGroups

            [in]One or more group ids (bitwise or-ed, 1, 2, 4, 8, …… 0x80000000)

 

UQueue

            [in]A memory buffer containing a message to be sent

 

MessageBuffer

            [in]A memory buffer containing a message to be sent

 

pBuffer

            [in]A memory buffer containing a message to be sent

 

nLen

            [in]Size of Buffer (pBuffer) in byte

 

Remarks

 

Calling this method sends a message in array of bytes to one or more chat groups.

 

Refer to the methods Speak of the interface IUChat and SpeakEx of the interface IUFast.
