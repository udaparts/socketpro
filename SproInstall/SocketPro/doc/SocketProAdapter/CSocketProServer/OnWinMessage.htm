protected virtual void OnWinMessage(int hWnd, int nMessage, int wParam, int lParam)

protected virtual bool OnWinMessage(int hWnd, int nMessage, int wParam, int lParam, int dwTime, int nPointX, int nPointY)

 

Parameters

 

hWnd

            [in]A window handle

 

nMessage

            [in]A message

 

wParam

            [in]The first parameter

 

lParam

            [in]The second parameter

 

dwTime

            [in]Time tick count at this time

 

nPointX

            [in]An x value

 

nPointY

            [in]A y value

 

Remarks

 

This virtual function is called with main thread when a message pump in SocketPro server detects a message. For details, you may refer to the window system functions GetMessage, PeerMessage, DispatchMessage and structure MSG.

 

For the 1st version of this virtual function, refer to the native method SetOnWinMessage and callback POnWinMessage.
