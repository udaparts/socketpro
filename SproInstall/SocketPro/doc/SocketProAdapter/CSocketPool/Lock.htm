public THandler Lock(int nTimeout, System.IntPtr pIUSocketSameThread)

 

Parameters

 

nTimeout

            [in]An integer indicating timeout in ms. It defaults to –1 or 0xFFFFFFFF

 

pIUSocketSameThread

            [in]An interface to a COM USocket object in usocket.dll

           

Remarks

 

Calling the function locks an asynchronous handler. If failed, the function returns null. Otherwise, the function returns a reference to an asynchronous handler.

 

If the input parameter pIUSocketSameThread is a valid pointer to a COM USocket object, the function will lock a new COM USocket object that is hosted within the same thread as pIUSocketSameThread.
