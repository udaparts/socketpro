public bool StartSocketPool(string strHost, int lPort, string strUID, string strPassword, byte bSocketsPerThread)

public bool StartSocketPool(string strHost, int lPort, string strUID, string strPassword, byte bSocketsPerThread, byte bThreads, USOCKETLib.tagEncryptionMethod EncrytionMethod, bool bZip)

 

Parameters

 

strHost

            [in]An IP address to a remote SocketPro server

 

lPort

            [in]A remote SocketPro server port number

 

strUID

            [in]A user id

 

strPassword

            [in]A password

 

bSocketsPerThread

            [in]The number of USocket objects hosted within a thread

 

bThreads

            [in]The number of thread within a COM USocketPool object. It defaults to the number of CPU processors

 

EncryptionMethod

            [in]Encryption method used for socket communication. It defaults to NoEncryption

 

bZip

            [in]A Boolean value indicating if online compression is turned on           

 

Remarks

 

Calling the function starts a pool of sockets and tries to connect to remote SocketPro server. If successful, all of pooled sockets will initially switched to the service of THandler.

 

Note that the maximum number of sockets in a pool is 64. If the input parameters bSocketsPerThread and bThreads are not correct or proper, CSocketPool will internally adjust them automatically.
