public long Diff(long lPrevCount)

public long Diff(long lNow, long lPrevCount)

 

Parameters

 

lPrevCount

            [in]High-resolution performance count obtained previously

 

lNow

            [in]High-resolution performance count obtained at this time

 

Remarks

 

Calling one of the two methods gets the difference of obtained performance counts between this time and previously. You can call the first method after the method Now. The two methods return the time difference in micro-second.
